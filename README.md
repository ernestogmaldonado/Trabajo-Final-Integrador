# TP Final - Gestion de Proyectos

Trabajo con NestJS + Angular + PostgreSQL.

## Como correrlo

1. Instalar PostgreSQL y crear la base `gestion_proyectos` (usuario y clave: postgres).
2. Ejecutar **iniciar.bat** (o en dos consolas: `npm run start:dev` en backend y `npm start` en frontend).
3. Entrar a http://localhost:4200
4. Usuario: **admin** / Clave: **admin123**

La API queda en http://localhost:3000/api

**Importante:** el puerto 3000 tiene que estar libre (si tenes MySQL u otro programa ahi, detenerlo antes).
